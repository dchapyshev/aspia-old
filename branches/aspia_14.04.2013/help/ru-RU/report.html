Сохранение отчета

Во многих случаях требуется сохранить полученные данные для последующего их просмотра на другом компьютере или для сравнения изменений в разные промежутки времени. Для этих целей Aspia имеет средства для сохранения собранных данных в файлы нескольких форматов. Так же существует несколько способов сохранения отчета.
В главном окне программы на панели инструментов имеется кнопка «Сохранить отчет», при нажатии на которую выводится меню с вариантами сохранения (рис. 1).

Рис. 1

Пользователь может выбрать сохранение:
  • Текущей открытой страницы
  • Всех категорий
  • Выборочное сохранение категорий

При выборе пунктов меню «Текущая страница» или «Все категории» пользователю выводится стандартное диалоговое окно сохранения файла, где он может выбрать желаемый формат файла имя файла и директорию, в которую отчет будет сохранен.

После нажатия кнопки «Сохранить» будет запущен процесс сохранения отчета, состояние которого отображается в соответствующем диалоговом окне (рис. 2).

Рис. 2

Выборочное сохранение отчета позволяет определить список категорий, которые будут сохранены в файл. Диалог выборочного сохранения (рис. 3) позволяет пользователю указать формат, имя и путь к файлу, а так же определить необходимость добавления оглавления в отчет.

Рис. 3


При нажатии кнопки «Сохранить» будет запущено сохранение отчета с отображением текущего состояния (рис. 2).
