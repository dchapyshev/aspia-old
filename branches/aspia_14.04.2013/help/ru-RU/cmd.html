Работа из командной строки

Aspia может работать из командной строки. Эта возможность позволяет сохранить отчет в выбранном формате и в указанную пользователем директорию без запуска графического интерфейса.

aspia.exe [file_path].[file_ext] [-nonav] [-help]

file_path - путь к файлу и имя файла. Например, C:\Temp\report.htm
file_ext - расширение файла. Необходимо правильно указать расширение, так как по нему определяется формат файла, в который будет сохранен отчет. Поддерживаются следующие расширения (форматы):

  • htm - HTML-файл
  • txt - Текстовый файл
  • csv - Таблица с разделителем ";"
  • rtf - Текст в формате Rich Text Format
  • ini - INI-файл
-nonav - параметр отключает добавление оглавления в файл отчета.
-help - выводит справку по использованию Aspia в командной строке.

Примеры использования:
  • Вывод справки:
    aspia.exe -help
  • Сохранение отчета без оглавления в директорию C:\Temp с именем файла test в формате HTML:
    aspia.exe C:\Temp\test.htm -nonav
  • Сохранение отчета с оглавлением в директорию C:\Temp с именем файла test2 в формате RTF:
    aspia.exe C:\Temp\test2.rtf
