Горячие клавиши

Aspia поддерживает горячие клавиши, которые позволяют вызвать некоторые функции программы не прибегая к помощи мыши.

Поддерживаются следующие горячие клавиши:

  • Ctrl + Q - Выход их программы.
  • Ctrl + S - Сохранение отчета.
  • Ctrl + A - Выделить все строки в окне информации.
  • Ctrl + C - Копирование выделенной строки (или строк).
  • F5 - Обновление текущей открытой страницы.
  • F1 - Вызов справки по использованию программы.
